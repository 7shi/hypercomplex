# ホップファイブレーション

ホップファイブレーションに関する記事です。

## 四元数とホップファイブレーション

1. [四元数からの生成](01-quaternion.md) - Wikipediaで説明されている四元数からの生成方法を確認します。
2. [スピノルのテンソル積](02-spinor-tensor.md) — 回転子の成分を具体的に計算し、位相差が内積・外積に対応することを確認します。状態ベクトルとパウリ行列によるブロッホベクトルの導出、テンソル積から密度行列が得られることも見ます。

## 検証コード

プロジェクトルートで `uv run hopf/<ファイル名>` により実行します。

- [check-01-quaternion.py](check-01-quaternion.py) — $ω\mathbf kω^*$の成分展開とノルム、複素数ペア・パウリ行列による表式の一致、ファイバー$q=u+v\mathbf k$の検証。
- [check-02-spinor-tensor.py](check-02-spinor-tensor.py) — $ωq$による$\mathbf k$項の消去、三角関数表示での回転結果、ブロッホベクトルと密度行列の検証。01-quaternion.md の$ω$との対応（右から$-\mathbf k$を掛けたもの）も確認します。

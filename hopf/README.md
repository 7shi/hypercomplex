# ホップファイブレーション

ホップファイブレーションに関する記事です。

## その他の記事

- [複素2次元ベクトルから2次元球面への射影](c2-to-s2.md) — 正規化された複素数のペアから、大域位相の無視と赤道の1点への縮退という2段階の同値関係を経て2次元球面上の座標へ至る過程を初等的に説明し、ホップ写像の式を導きます。
- [同次座標とホップファイブレーション](homogeneous.md) — 3次元球面上の実4次元座標を複素数のペアの比（同次座標）に変換し、立体射影によって2次元球面へ写す計算を通じてホップファイブレーションを構成します。

## 四元数とホップファイブレーション

1. [四元数からの生成](01-quaternion.md) - Wikipediaで説明されている四元数からの生成方法を確認します。
2. [スピノルのテンソル積](02-spinor-tensor.md) — 回転子の成分を具体的に計算し、位相差が内積・外積に対応することを確認します。状態ベクトルとパウリ行列によるブロッホベクトルの導出、テンソル積から密度行列が得られることも見ます。
3. [ブロッホ球と密度行列](03-bloch-density.md) — パウリ行列を基底として純粋状態・混合状態を密度行列で表し、混合と重ね合わせの違い、測定の期待値との関係まで見ます。
4. [四元数・八元数への拡張](04-extension.md) — 複素数によるホップファイブレーションを四元数$S^7\to S^4$・八元数$S^{15}\to S^8$に拡大し、対応する写像を構成します。非結合性がどこで効くかを確認します。

## 検証コード

プロジェクトルートで `uv run hopf/<ファイル名>` により実行します。

- [check-01-quaternion.py](check-01-quaternion.py) — $ω\mathbf kω^*$の成分展開とノルム、複素数ペア・パウリ行列による表式の一致、ファイバー$q=u+v\mathbf k$の検証。
- [check-02-spinor-tensor.py](check-02-spinor-tensor.py) — $ωq$による$\mathbf k$項の消去、三角関数表示での回転結果、ブロッホベクトルと密度行列の検証。01-quaternion.md の$ω$との対応（右から$-\mathbf k$を掛けたもの）も確認します。
- [check-03-bloch-density.py](check-03-bloch-density.py) — パウリ行列の積とエルミート行列の展開、純粋状態の性質（べき等性・位相不変性）、混合状態のブロッホベクトルと純粋度、固有分解、混合と重ね合わせの例の検証。
- [check-04-extension.py](check-04-extension.py) — $H(\alpha, \beta)=(2\alpha\beta^*, |\alpha|^2-|\beta|^2)$ の四元数・八元数への拡張によるノルム1の保存の検証、および右乗算によるファイバーが四元数では結合性により保存され、八元数では非結合性により破壊されることの確認。
- [check-04-nonassoc-fiber.py](check-04-nonassoc-fiber.py) — 八元数で右乗算がファイバーを保存しない具体例（$\alpha=e_1/\sqrt2$等）と、左乗算$\beta=-p\alpha$によるファイバーの構成、一般の点$(c, r)$への拡張の検証。

## 方針

現時点ではパウリ行列・トレースなど標準的な計算手法を踏襲していますが、いずれ全体を幾何代数の手法で書き直す構想があります。

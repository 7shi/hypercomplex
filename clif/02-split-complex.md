分解型複素数は、虚数単位 $j$ が $j^2=+1$ を満たす特殊な複素数体系です。その代数構造は冪等元と極小イデアルを通じて理解されます。 本ページでは、分解型複素数の定義から始め、冪等元の直交性や零因子としての性質を探究し、その代数的性質と構造について解説します。

# 分解型複素数の定義

分解型複素数は、通常の複素数を拡張した数学的概念です。2つの実数 $x$ と $y$、そして $j^2 = +1$ を満たす実数でない虚数単位 $j$ を用いて、以下の形式で定義されます。

&&&def 分解型複素数 $z$
$$
z = x + jy
$$
&&&

ここで、$x$ と $y$ は実数であり、$j$ は「虚数単位」と呼ばれますが、通常の複素数の虚数単位 $i$ とは異なり、$j^2 = +1$ という性質を持ちます。この性質が、分解型複素数を通常の複素数と区別する重要な特徴となります。

分解型複素数全体の集合は「分解型複素平面」（split-complex plane）と呼ばれます。この平面上での演算は以下のように定義されます。

1. 加法：$(x_1 + jy_1) + (x_2 + jy_2) = (x_1 + x_2) + j(y_1 + y_2)$

2. 乗法：$(x_1 + jy_1)(x_2 + jy_2) = (x_1x_2 + y_1y_2) + j(x_1y_2 + y_1x_2)$

これらの演算は可換的かつ結合的であり、加法に対して分配法則が成り立ちます。

また、分解型複素数は代数学的には、正定値二次形式を備えた一次元ベクトル空間上のクリフォード代数として解釈することができます。これは、通常の複素数が負定値二次形式を備えた一次元ベクトル空間上のクリフォード代数であることと対照的です。

# $j$の行列表現

$j^2 = +1$ を満たす実数でない虚数単位は、2×2の実行列として表現できます。ここでは固有値 $\pm1$ がそのまま対角に並ぶ対角行列を採用します。

$$
j \cong \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix},
\quad
j^2 \cong \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}^2 = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix} = I \cong 1
$$

この表現では、一般の分解型複素数も対角行列に対応します。

$$
z = x + jy \cong \begin{pmatrix} x + y & 0 \\ 0 & x - y \end{pmatrix}
$$

対角行列の積は対角成分ごとの積になるため、2つの成分 $x \pm y$ は互いに干渉せず、独立に振る舞います。この2つの成分が、以下で見る代数構造の全体を通じて中心的な役割を果たします。

&&&rem 非対角の行列表現
$j$ は非対角の行列で表現することもできます。

$$
j \cong \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix},
\quad
j^2 \cong \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}^2 = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix} = I \cong 1
$$

この行列の固有値も $\pm1$ で、対角化すれば本文の対角行列が得られます。両者は基底の取り替えで結ばれた同等な表現です。
&&&

# 分解型複素数の冪等元

分解型複素数環における冪等元は、その代数構造を特徴づける重要な要素です。冪等元とは、自身との積が自身に等しくなる元のことを指します。分解型複素数環 $\mathbb{R}[j]$ において、主要な冪等元は以下の2つです。

&&&def 冪等元 $e$ と $e^*$
$$
e = \frac{1 - j}{2}, \quad e^* = \frac{1 + j}{2}
$$
&&&

これらの冪等元は以下の性質を満たします。

1. 冪等性：$e^2 = e, \quad (e^*)^2 = e^*$
2. 直交性：$ee^* = e^*e = 0$
3. 完全性：$e + e^* = 1$

これらの性質は、分解型複素数環の構造を理解する上で非常に重要です。

行列表現では、$e$ と $e^*$ は対角成分の一方だけを取り出す射影行列になります。

$$
e \cong \begin{pmatrix} 0 & 0 \\ 0 & 1 \end{pmatrix},
\quad
e^* \cong \begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix}
$$

冪等性・直交性・完全性は、いずれもこの対角行列の計算として一目で確認できます。

&&&rem 固有空間としての見方
「$j$ を掛ける」操作は、分解型複素数環を実2次元の線形空間（基底 $1, j$）とみなしたときの線形写像です。冪等元はこの線形写像の固有ベクトルであり

$$
je = \frac{j - j^2}{2} = -e,\quad je^* = \frac{j + j^2}{2} = e^*
$$

より、固有値はそれぞれ $-1$ と $+1$ です。固有値 $-1$ の固有空間は $e$ の実数倍全体 $\mathbb{R}e$、固有値 $+1$ の固有空間は $\mathbb{R}e^*$ で、それぞれ1次元です。本文の対角行列による表現は、この固有ベクトルを基底に取ることに相当し、だからこそ $j$ の行列表現には固有値 $\pm1$ が対角に並びます。
&&&

冪等元の存在は、分解型複素数環が通常の複素数体とは異なる代数的性質を持つことを示しています。特に、0でも1でもない非自明な冪等元が存在することは、分解型複素数環の特徴的な性質です。

以上のように、冪等元は分解型複素数環の代数的構造を特徴づける中心的な概念であり、その性質を理解することは、この数体系を扱う上で不可欠です。

# 冪等元と零因子

分解型複素数環において、冪等元は零因子としての性質も持ちます。零因子とは、環の中で0でない元同士の積が0になるような元のことを指します。主要な冪等元 $e$ と $e^*$ は、以下の関係を満たすことから零因子であることがわかります。

$$
ee^* = e^*e = 0
$$

この性質は、分解型複素数環の特徴的な代数構造を反映しています。通常の複素数体では零因子が存在しないのに対し、分解型複素数環では非自明な零因子が存在することが大きな違いです。

さらに、任意の非自明な冪等元 $a \neq 0, 1$ は常に零因子となります。これは以下のように示すことができます。

$$
(1 - a)a = a - a^2 = a - a = 0
$$

ここで、$a \neq 0$ かつ $1 - a \neq 0$ であることから、$a$ は零因子であることがわかります。

この性質は、分解型複素数環が整域（零因子を持たない可換環）ではないことを示しています。これは、通常の複素数体とは異なる代数的性質であり、分解型複素数を用いた数学的モデルや応用において重要な役割を果たします。

# 冪等元の直交性

分解型複素数環における冪等元の直交性は、その代数構造を特徴づける重要な性質です。2つの冪等元 $e$ と $f$ が直交するとは、$ef = fe = 0$ が成り立つことを意味します。

分解型複素数環の場合、主要な冪等元である $e$ と $e^*$ は直交しています。具体的には、

$$
ee^* = e^*e = \left(\frac{1 - j}{2}\right)\left(\frac{1 + j}{2}\right) = \frac{1 - j^2}{4} = 0
$$

となります。この直交性は、分解型複素数環の構造を理解する上で重要な役割を果たします。

直交する冪等元の和に関しても興味深い性質があります。$e$ と $f$ が直交する冪等元である場合、$e + f$ もまた冪等元となります。これは以下のように示すことができます。

$$
(e + f)^2 = e^2 + ef + fe + f^2 = e + 0 + 0 + f = e + f
$$

ここで、$e^2 = e$ と $f^2 = f$ （冪等性）、および $ef = fe = 0$ （直交性）を用いています。

この直交性の概念は、分解型複素数環に限らず、より一般的な環論においても重要です。特に、von Neumann正則環や自己入射的な環では、直交する冪等元の性質がその環の構造を特徴づける重要な要素となります。

# 冪等元と極小イデアル

分解型複素数環における冪等元と極小イデアルの関係は、その代数構造を理解する上で重要な役割を果たします。冪等元 $e$ と $e^*$ は、それぞれ極小イデアルを生成します。

&&&def イデアル
環の部分集合であって、加法について閉じており、環のどの元を掛けてもその部分集合の中に留まる（吸収性を持つ）ものを**イデアル**と呼びます。
&&&

&&&def 極小イデアル
非零イデアルであって、自分自身以外の非零イデアルを含まないものを**極小イデアル**と呼びます。
&&&

分解型複素数環 $\mathbb{R}[j]$ において、$Re = \{re \mid r \in \mathbb{R}\}$ と $Re^* = \{re^* \mid r \in \mathbb{R}\}$ はそれぞれ極小イデアルです。

&&&prf $Re$ がイデアルであること
$je = \dfrac{j - j^2}{2} = -e$ より、任意の元 $x + jy \in \mathbb{R}[j]$ を掛けると

$$
(x + jy)(re) = r(xe + yje) = r(x - y)e \in Re
$$

となり、$Re$ の中に留まります（$Re^*$ についても $je^* = e^*$ より同様です）。
&&&

&&&prf $Re$ が極小イデアルであること
イデアルは実数倍と加法について閉じているため、$\mathbb{R}[j]$ の部分ベクトル空間になります。$Re$ は $e$ の実数倍全体からなる1次元の部分空間なので、その中に真に含まれる部分空間は $\{0\}$ しかなく、$Re$ より小さい非零イデアルは存在しません。したがって $Re$ は極小イデアルです。
&&&

冪等元と極小イデアルの関係は、分解型複素数環の特徴的な代数構造を反映しており、通常の複素数体とは異なる性質を示しています。この構造は、分解型複素数を用いた計算や理論的な解析において、重要な役割を果たします。

# 冪等元による直和分解

分解型複素数環において、冪等元の直交性はイデアルの分離を引き起こす重要な性質です。この性質は、環の代数構造をより深く理解する上で重要な役割を果たします。

主要な冪等元 $e$ と $e^*$ の直交性により、分解型複素数環 $\mathbb{R}[j]$ は2つの極小イデアルの直和として表現できます。

$$
\mathbb{R}[j] = Re \oplus Re^*
$$

ここで、$Re$ と $Re^*$ はそれぞれ $e$ と $e^*$ によって生成される極小イデアルです。この分解は以下のように示すことができます。

1. 直交性：$ee^* = e^*e = 0$ であることから、$Re$ と $Re^*$ の積は0になります。

2. 完全性：任意の元 $z = x + jy \in \mathbb{R}[j]$ は、以下のように $Re$ と $Re^*$ の元の和として表現できます。

   $z = (x-y)e + (x+y)e^*$

   これは、$e + e^* = 1$ という性質を利用しています。

3. 直和性：$Re \cap Re^* = \{0\}$ であることは、$ae = be^*$ ($a,b \in \mathbb{R}$) ならば $a = b = 0$ であることから示されます。

線形代数の言葉では、この直和分解は「$j$ を掛ける」写像の固有空間分解に他なりません。極小イデアル $Re$ と $Re^*$ は、それぞれ固有値 $-1$ と $+1$ の固有空間です。

この性質は、分解型複素数環が半単純環であることを示しています。半単純環とは、極小イデアルの直和として表現できる環のことを指します。この構造は、通常の複素数体とは異なる分解型複素数環の特徴的な性質を反映しています。

この冪等元の直交性によるイデアルの分離は、分解型複素数環の代数的構造を特徴づける重要な性質であり、この環を用いた数学的モデルや応用において重要な役割を果たします。

# 直和分解と積の簡略化

分解型複素数環における冪等元による直和分解は、乗法演算を大幅に簡略化します。任意の分解型複素数 $z_1 = ae + be^*$ と $z_2 = ce + de^*$ の積は、冪等元 $e$ と $e^*$ の性質を利用して以下のように計算できます。

$$
(ae + be^*)(ce + de^*) = ace^2 + ade^*e + bce^*e + bde^{*2} = ace + bde^*
$$

この簡略化は、冪等元の性質 $e^2 = e$, $(e^*)^2 = e^*$, および直交性 $ee^* = e^*e = 0$ に基づいています。行列表現では $ae + be^* \cong \begin{pmatrix} b & 0 \\ 0 & a \end{pmatrix}$ であり、この簡略化は対角行列の積が対角成分ごとの積になることに対応します。この結果は、分解型複素数環が実数環の直和 $\mathbb{R} \oplus \mathbb{R}$ と同型であることを反映しており、各成分が独立に乗算されることを示しています。この簡略化された乗法は、分解型複素数を用いた計算を効率化し、その代数的構造をより明確に理解するのに役立ちます。

# まとめ

本記事では、分解型複素数環 $\mathbb{R}[j]$（$j^2 = +1$）を冪等元 $e = (1-j)/2,\ e^* = (1+j)/2$ を通じて特徴づけました。この2つの冪等元は互いに直交し（$ee^* = e^*e = 0$）、和が1になる（$e + e^* = 1$）という性質を持ち、非自明な冪等元として分解型複素数環が零因子を持つこと、すなわち整域ではないことを示しています。この直交性・完全性により、環全体は2つの極小イデアル $Re, Re^*$ の直和 $\mathbb{R}[j] = Re \oplus Re^*$ として半単純環の構造を持つことが分かり、任意の元は $z = (x-y)e + (x+y)e^*$ と一意に分解できます。この分解のもとで乗法は成分ごとの積に帰着し、分解型複素数環が実数環の直和 $\mathbb{R} \oplus \mathbb{R}$ と同型であることが確認できます。この直和分解は「$j$ を掛ける」写像の固有空間分解でもあり、冪等元 $e, e^*$ はその固有ベクトル（固有値 $\mp1$）です。この構造は行列表現 $j \cong \operatorname{diag}(1, -1)$ に集約されており、冪等元は射影行列 $e \cong \operatorname{diag}(0, 1),\ e^* \cong \operatorname{diag}(1, 0)$、任意の元は対角行列 $z \cong \operatorname{diag}(x+y,\ x-y)$ として、すべての性質が対角成分ごとの計算に帰着します。

# クリフォード代数

クリフォード代数に関する記事です。

## 記事

1. [クリフォード代数の構造と表現](01-representation.md) — 実数体・複素数体・四元数体・分解型代数を基本構造とし、テンソル積による拡張公式を用いてクリフォード代数を行列環として分類します。分類表から擬スカラーによる型の判別、偶部分代数、複素化、ピノル表現を読み取ります。
2. [分解型複素数の冪等元と直和分解](02-split-complex.md) — 分解型複素数を冪等元によって特徴づけ、零因子性・直交性・極小イデアルとの関係から、実数の直和への分解と乗法の簡略化を導きます。
3. [一般化パウリ行列と一般化クリフォード代数](03-gpm-gca.md) — パウリ行列をクロック行列・シフト行列として一般の次数に拡張し、九元数との対応を確認します。一般化クリフォード代数を定義し、生成元の追加が行列環とのテンソル積に対応することを示します。生成元が奇数個の場合は中心元による直和分解を導き、1個の場合としてフルーリーの多重複素数に触れます。
4. [クロック行列とシフト行列の連続極限：ワイル代数と正準交換関係](04-weyl-algebra.md) — クロック行列とシフト行列を円周上の格子に対する位相掛けと平行移動として解釈し直し、ねじれた交換関係がワイル関係式の厳密な有限次元実現であることを示します。掛け算と微分の交換関係$[Q,P]=i$は有限次元では実現できず（トレースの障害）、連続極限でハイゼンベルク代数・ワイル代数に移行します。
5. [離散ラプラシアンの連続極限：シュレーディンガー方程式の導出](05-schrodinger.md) — ノルムを保つ時間発展がエルミート行列の生成する$i\,d\psi/dt=H\psi$に限られることを示し、シフト行列の2階差分（離散ラプラシアン）とクロック行列の多項式（ポテンシャル）からハミルトニアンを構成します。連続極限でシュレーディンガー方程式を導き、正準交換関係が平均値の運動をニュートンの運動方程式の形に定めること（エーレンフェストの定理）を確認します。

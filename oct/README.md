# 八元数

八元数とその先の十六元数に関する記事と検証コードです。

## 記事

1. [八元数の基本事項](01-octonion.md) — 八元数の基本構造（8次元・非可換・非結合的なノルム多元体）、共役とノルムの乗法性、グレイブスとケイリーによる7つの三つ組、ファノ平面による乗法規則の視覚化、非結合性の具体例（三つ組内で閉じていれば結合的）について。
2. [単位八元数による7次元三重回転](02-7d-3rot.md) — 純虚八元数の積から定まる7次元の内積と外積、7つの三つ組による3次元外積の組み合わせ、単位八元数とその共役で挟む操作が生成する7次元回転（八元数版ロドリゲスの回転公式）、回転軸に直交する3つの2次元平面で同期した回転となる特異な構造について。
3. [八元数の左作用とクリフォード代数](03-oct-left-mul.md) — 八元数の左作用の行列表現。左作用が生成するクリフォード代数$C\ell_{0,6}(\mathbb R)\cong M_8(\mathbb R)$の構造、右作用の左作用による表現、スピノルへの射影$P$とその復元、非結合性が見せる構造について。
4. [十六元数と零因子](04-sedenion.md) — ケイリー＝ディクソン構成を八元数の先へ進めた十六元数。ノルムの乗法性の破れが八元数の結合子で記述されること、零因子の特徴づけと相方の空間、積が$0$になる単位ノルムのペアの全体が基本三つ組の空間すなわち例外型リー群$G_2$と同相になること（モレノの定理）、自己同型群$G_2 \times S_3$について。

## その他の記事

- [八元数の非結合性](nonassociativity.md) — 八元数の基本規則から出発し、結合法則を仮定すると矛盾が生じることを示すことで非結合性を導きます。反結合性、交代性、四元数と同型な三つ組内では結合的になることも整理します。

## 検証コード

プロジェクトルートで `uv sync` により環境を構築し、`uv run oct/<ファイル名>` で実行します。`octonion.py`は各検証スクリプトが共通で使う八元数の積演算モジュールです。

- [check-03-lxp-recovery.py](check-03-lxp-recovery.py) — 復元式 $L_x = \sum_{j=0}^{7} R_j (L_x P) R_j^{-1}$ の数値検証。各項が$L_x$の第$j$列を再現すること、右作用$R_1$が左作用の積の線形結合で書けることも確認します。
- [check-03-p-fano-group.py](check-03-p-fano-group.py) — 射影行列$P$の8項（$I$とファノ7直線の三重積）が位数8の可換群を成すことの数値検証。群の閉性、単位元の固定、共点3直線の乗積規則、剰余類$L_kG$による64基底の分類を確認します。
- [check-03-l1l2p-to-l3.py](check-03-l1l2p-to-l3.py) — $L_1L_2 \ne L_3$ の具体例で、射影$L_1L_2P = L_3P$と復元式による$L_3$の再構成を行列表示で確認します。
- [check-03-p-inline-expand.py](check-03-p-inline-expand.py) — 行列を使わない代数計算のシミュレート。$L_7$を記号のまま反交換で扱い、$P$をインライン展開した$L_1L_2P$と$L_3P$が同じ8項（分類表の$e_3$行）に一致することを項別に確認します。4文字以上の語は体積関係式$L_1\cdots L_7=-I$で3文字以下の補集合に畳み込みます。
- [check-03-8lip.py](check-03-8lip.py) — $8L_xP$を構成する8つの項が個別に単位元$1$を$e_x$へ送ることの検証。$PP \mapsto 1$および$L_xP \mapsto e_x$の項ごとの裏付けと、8本の等式$8L_xP = \dots$の確認を行います。
- [check-03-liri-form.py](check-03-liri-form.py) — 射影$P$を四元数と同様の$L_iR_i$形式で書けるかの検証。$L_iR_i$が結合性によらず対角行列（位置$0$と$i$が$-1$、他が$+1$）になることを確かめ、生成元の個数$n$についての一般公式$a=\frac{1}{2-2n},\,a_0=(2-n)a$を導いて、四元数（$n=3$）で係数がそろうのは偶然であり、八元数（$n=7$）では$P=\frac{1}{12}(5I-\sum_{i=1}^7 L_iR_i)$と不揃いになることを確認します。
- [check-04-sedenion.py](check-04-sedenion.py) — 04-sedenion.md の数式の数値検証。十六元数の基底の積規則（35個の三つ組・添字のXOR規則）、共役の反自己同型性と$xx^* = |x|^2$・両側逆元、冪結合性と柔軟法則、交代性の破れ、ノルムの乗法性の破れの公式$|xy|^2 = |x|^2|y|^2 + 2\,(d,a,c)\cdot b$と四元数の倍加では乗法性が保たれること、零因子$(e_1+e_{10})(e_5+e_{14})=0$と消去律の崩壊、モレノの特徴づけ（成分が直交する等ノルムの純虚ペア）と4次元の零化空間、基本三つ組$(u_1,u_2,u_3)$と零因子ペアの対応、$G_2$の対角持ち上げが十六元数の自己同型としてペアの空間に作用すること（固定部分群が自明・接空間14次元・軌道が全体を張ること）、$\dim\operatorname{Der}(\mathbb S) = 14$とすべてが対角持ち上げであること、離散自己同型$\varepsilon, \psi$と$S_3$の関係式を確認します。

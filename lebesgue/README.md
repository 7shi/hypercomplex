# ルベーグ積分

ルベーグ積分について、数学的厳密さよりも直観性を重視して説明する記事シリーズです。

## 記事

1. [測度とルベーグ積分](01-measure-integral.md) — 「極限と積分の交換の保証」という理論の目的を先に掲げ、リーマン積分の限界（ディリクレ関数）、「横切り」の発想と測度、単関数による積分の構成を経て、目的地である単調収束定理・優収束定理に到達します。応用例として$\int_0^1 x^{-x}\,dx = \sum 1/n^n$（sophomore's dream）を項別積分で導きます。
2. [L²空間とヒルベルト空間](02-l2-hilbert.md) — 可積分関数の空間がルベーグ積分によって初めて完備になること（リース＝フィッシャーの定理）を軸に、関数の内積と直交性からヒルベルト空間に進みます。フーリエ級数を「正規直交基底による展開」として捉え、パーセバルの等式をピタゴラスの定理の無限次元版として導き、直交射影定理とリースの表現定理まで整備します。応用例としてパーセバルの等式からバーゼル問題$\sum 1/n^2 = \pi^2/6$を導きます。
3. [測度論的確率論](03-probability.md) — 確率空間を全測度$1$の測度空間として定義し直し、確率変数＝可測関数、期待値＝ルベーグ積分という辞書を整備します。ボレルの正規数定理で辞書を実演した後、分布関数からルベーグ＝スティルチェス測度を構成して離散・連続の期待値公式を$E[X] = \int x\,dF(x)$に統一し、ラドン＝ニコディムの定理を経て条件付き期待値を$L^2$の直交射影として構成し、マルチンゲールに至ります。応用例として任意停止定理からギャンブラーの破産確率$b/(a+b)$と平均決着時間$ab$を導きます。
4. ブラウン運動と伊藤の補題（予定） — ブラウン運動の経路が有界変動でないため、経路ごとのルベーグ＝スティルチェス積分が不可能であることを確認し、伊藤の等長性と$L^2$の完備性によって伊藤積分を構成します。2次変分$(dB)^2 = dt$から伊藤の補題の補正項を導き、確率微分方程式への入口に立ちます。

## 構想

シリーズの最終目的地は**伊藤の補題**です。ルベーグ積分から出発して、$L^2$空間・測度論的確率論を経由し、確率解析に到達する一本道を構成します。

伊藤積分とルベーグ積分の関係は二重です。まず否定的な側面として、ブラウン運動の見本経路は至るところ微分不可能で有界変動ですらないため、$\int f\,dB$を経路ごとのルベーグ＝スティルチェス積分として定義することは原理的に不可能です。伊藤積分が独自の理論として必要なのは、ルベーグ流の積分がそこで通用しないからです。一方で肯定的な側面として、伊藤積分の構成は「単純な過程に積分を定義し、極限で拡張する」というルベーグ積分の構成（指示関数 → 単関数 → 一般の関数）の忠実な反復であり、その極限の存在を保証するのが伊藤の等長性と$L^2$の完備性です。また、伊藤の補題の補正項$\frac{1}{2}f''\sigma^2\,dt$は2次変分$(dB)^2 = dt$の帰結で、これは経路が有界変動でないことの裏返しです。

この目的地から逆算すると、残りの記事の役割が定まります。03までは執筆済みで、収束定理（01）、$L^2$の完備性・直交射影定理・リースの表現定理（02）、条件付き期待値・マルチンゲール・ルベーグ＝スティルチェス積分（03）という道具が揃っています。

- **04（ブラウン運動と伊藤の補題）**: 上記の二重の関係を主題とし、シリーズ全体の道具立てを総動員して伊藤積分を構成し、伊藤の補題に到達します。2次変分$(dB)^2 = dt$から補正項を導き、確率微分方程式への入口に立ちます。構成の型は02で見た完備化そのものです。すなわち、単純な過程の上で積分を定義し、伊藤の等長性で$L^2$への等長対応を作り、完備性で全体に延長します。03で用意した布石はそれぞれ、否定的主張（経路が有界変動でないため経路ごとのスティルチェス積分は不可能）、被積分過程の非予見性（マルチンゲール変換の$\mathcal{F}_{n-1}$可測な賭け金の連続版）、補正項の源泉（$S_n^2 - n$がマルチンゲール＝2乗の決定論的成長）として回収します。

02の執筆で得た教訓を2つ記録しておきます。第一に、a.e.同一視の下では見かけの反例が消えることがあります。当初「01の関数列$f_n \to \mathbb{1}_{\mathbb{Q}}$が完備性の破れを示す」と構想していましたが、同一視の下では列も極限もすべてゼロベクトルであり、穴の証拠になりません。実際の記事では$\varepsilon/2^n$被覆で有理数を太らせた開集合の指示関数を構成しました。04の否定的主張（経路ごとの積分は不可能）を述べる際も、測度$0$の修正で救えないことまで含めて確認する必要があります。第二に、締めくくりの型です。各記事は「理論は値の計算には登場せず、等式の保証だけを与える」ことを実演する応用例で締めます（01はsophomore's dream、02はバーゼル問題）。03・04でも同じ型の締めを用意します。

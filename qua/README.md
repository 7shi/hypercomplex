# 四元数

四元数に関する記事です。

## 記事

1. [パウリ行列と四元数：双四元数による橋渡し](01-pauli-qua.md) — パウリ行列と四元数の関係。双四元数を導入して両者を統一的に理解し、パウリ行列が生成する代数構造がクリフォード代数の一部であることを示します。
2. [シルベスターの九元数 (nonion)](02-nonion.md) — シルベスターの「九元数 (nonion)」の論考の再構成。$2$次行列から四元数が現れる流れ、行列単位による記述、四元数から九元数への拡張について。
3. [四元数のテンソル積とクリフォード代数の構成](03-qua-tensor.md) — 四元数と分解型四元数のテンソル積からクリフォード代数を構成し、生成元の選択と計量・符号数の関係を分析します。両者のテンソル積が同型になることを示し、拡張時の符号数変化を一般公式にまとめます。
4. [4次元回転と分解型双四元数](04-4d-bsqua.md) — 単位四元数の左右からの挟み込みが$\mathbb H\otimes\mathbb H\cong M_4(\mathbb R)$の中で$\mathrm{SO}(4)$を直接構成する仕組みと、二重被覆$\operatorname{Spin}(4)$の置き場所である偶部分代数（分解型双四元数）への降下、回転子と左右の回転子の対応を辿ります。
5. [二重四元数と剛体変換](05-dual-qua.md) — 回転が微小領域で並進と見なせることを、$\varepsilon^2=0$を満たす形式的な基底$\varepsilon$によって厳密化します。二重数を係数に持つ二重四元数で、回転に加えて並進を含む剛体変換を挟み込みの形で統一的に表し、ねじ運動の指数表示、退化クリフォード代数$C\ell_{3,0,1}(\mathbb R)$の偶部分代数との同型を示します。
6. [回転子の補間：SLERP](06-slerp.md) — 2つの姿勢を結ぶ補間を、単位四元数のなす球面$S^3$上の測地線として構成します。なす角・対数写像・冪を整備してSLERPの冪形式とsin重み形式の等価性と等速性を示し、正規化線形補間 (NLERP) との比較、一般次元の回転子への拡張、二重四元数によるScLERPへの接続を扱います。

## その他の記事

- [四元数と球面三角法](spherical-trig.md) — 四元数の共役作用によって球面上の回転を表現し、回転を一周分合成すると符号が反転した恒等変換になることから、球面三角形の余弦定理・正弦定理・5要素の規則を導出します。
- [実パウリ行列とトレース](real-pauli-trace.md) — パウリ行列の関係式を実数のみで表現し直すことで実パウリ行列を構成し、単位行列とあわせて$2$次正方実行列の基底をなすことを示します。この基底での行列の積のトレースが係数ベクトルの内積の2倍に相当することを確認します。
- [複素数の拡張で学ぶテンソル積](tensor-from-complex.md) — 複素数に可換な虚数単位を追加すると双複素数が得られ零因子を持つことを確認し、非可換な積の演算子としてテンソル積を導入して、係数の可換性と基底の非可換性という規則によって双複素数の積と一致することを示します。
- [四元数の左作用・右作用とスピノルの左イデアル](spinor-ideal.md) — 四元数の左作用・右作用が生成する$C\ell_{3,1}(\mathbb R)\cong M_4(\mathbb R)$の中で、射影行列によってスピノル（四元数自身）を左イデアルとして埋め込み、右作用の共役和で演算子へ復元します。八元数の同種の構成との対比も述べます。

## 検証コード

プロジェクトルートで `uv sync` により環境を構築し、`uv run qua/<ファイル名>` で実行します。

- [check-04-4d-bsqua.py](check-04-4d-bsqua.py) — 04-4d-bsqua.md の数式の数値検証。$\omega$による2ベクトルの対と冪等元$\dfrac{1\pm\omega}2$による$\mathbb H\oplus\mathbb H$分解、対角表現の成分ごとの積と$T$の準同型性、等傾回転の合成による二重回転（$e^{\alpha i}qe^{\beta i}$の2平面の回転角$\alpha\pm\beta$）、正規形・一般の回転面での$r_L\,q\,r_R$とクリフォード代数の回転の一致、$L_uR_v$が張る$M_4(\mathbb R)\cong\operatorname{Cl}_{3,1}(\mathbb R)$、テンソル積での符号の相殺（2対1対応の核）、偶部分代数の階梯を確認します。
- [check-05-dual-qua.py](check-05-dual-qua.py) — 05-dual-qua.md の数式の数値検証。二重数の演算と$f(x+\varepsilon y)=f(x)+\varepsilon y f'(x)$、3種の共役と二重数値ノルム、剛体変換のサンドイッチ公式と$\mathrm{SE}(3)$の合成則（通常の共役では並進が消える負のチェックを含む）、接平面モデル（$\varepsilon$倍角の回転子による並進と基点を通る軸の回転）、ねじ運動の指数表示とねじ軸の抽出、$C\ell^+_{3,0,1}(\mathbb R)$との同型（全$64$積）、擬スカラーの三分法を確認します。
- [check-06-slerp.py](check-06-slerp.py) — 06-slerp.md の数式の数値検証。対数写像と冪の指数法則、なす角と半角関係、大円弧の等速性と最短性、冪形式とsin重み形式の一致（外挿を含む）、左右不変性と対称性、世界座標軸と角速度の一定性、符号選択と2本の経路（中間の姿勢が異なる負のチェック）、NLERPの不等速性、$C\ell^+_{3,0}(\mathbb R)$・$C\ell_{4,0}(\mathbb R)$での回転子の補間（線形結合が回転子にならない負のチェック）、ScLERPの端点・実部・軸上の並進、退化の数値安定性を確認します。

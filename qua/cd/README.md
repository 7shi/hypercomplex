# ケイリー＝ディクソン構成 (cd)

複素数のケイリー＝ディクソン構成に関する記事です。

## 記事

1. [複素数の行列表現から四元数へ：発見法的な導出](grok45.md) — 複素数の共役積による内積・外積の分解と、行列表現の成分を複素数化する操作を組み合わせ、四元数の行列表現をヒューリスティックに導出します。双複素数との分岐点や、リー群の記事の規約との対応も扱います。
2. [ケイリー＝ディクソン構成の分岐点：双複素数と四元数](bcmp-qua.md) — 複素数の対に対するケイリー＝ディクソン構成の積を展開し、虚数単位が複素数を追い越す規則の選び方によって可換な双複素数と非可換な四元数に分岐することを示します。それぞれの積公式から行列表現を組み立てます。
3. [四元数の実行列表現から複素行列表現へ](real-to-cmatrix.md) — 四元数の乗算規則から$4\times4$の実行列表現を構成し、成分の並べ替えによって$2\times2$複素行列表現に変換できることを示します。並び順を選ぶとパウリ行列にも対応することを確認します。
4. [ケイリー＝ディクソンの構成法から四元数の行列表現まで](cd-to-matrix.md) — ケイリー＝ディクソン構成の積公式を目標に据え、未知の行列成分を仮置きしては共役の不一致を修正していく試行錯誤によって、四元数の行列・ベクトル表現を導出します。
5. [発見的手法による四元数の複素行列表現の構成](heuristic-cmatrix.md) — 四元数を複素数の組として複素ベクトルへ素朴に対応させると$j$の作用が共役を含み線形変換にならないことを示し、成分の符号を反転する工夫によって乗算構造を保つ複素行列表現を得ます。
6. [四元数の行列表現からパウリ行列まで](matrix-to-pauli.md) — 複素数の実行列表現を固定し、四元数の反交換関係から残りの行列を連立方程式的に導出したのち、パウリ行列を構成します。エルミート共役と四元数の共役の対応も扱います。

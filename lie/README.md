# リー群・リー代数の初歩

リー群とリー代数について、数学的厳密さよりも直観性を重視して説明する記事シリーズです。

## その他の記事

- [代数的構造の初歩：マグマから群へ](magma-to-group.md) — 集合と二項演算の組を出発点に、閉性・結合性・単位元・逆元という性質を段階的に加えることで、マグマ・半群・モノイド・群という代数的構造が定義される過程を説明します。各性質を自然数・整数・実数の演算例で確認し、マグマから群へ至る別経路も紹介します。
- [二重被覆と単連結性の直観](double-cover.md) — シリーズで繰り返し現れる2重被覆がなぜ必然なのかを位相の側から説明します。$\operatorname{SO}(3)$を表面の対蹠点を貼り合わせたボールとして描き、回転1周のループが縮められないこと（単連結でないこと）と回転2周がほどけること（ベルトトリック）を、被覆への持ち上げで確かめます。

## 記事

1. [複素数とSO(2)](01-u1-so2.md) — 複素数から構成されるリー群$\operatorname{U}(1)$を例に、リー群とリー代数の基本概念（指数写像、基底、括弧積）を説明します。複素数の行列表現を通じて$\operatorname{SO}(2)$との同型を示します。
2. [四元数とSU(2)](02-su2-so3.md) — 四元数から構成されるリー群$\operatorname{Sp}(1) \cong \operatorname{SU}(2)$。非可換な括弧積が外積と一致することを見て、共役作用による$\operatorname{SO}(3)$の2重被覆とリー代数の同型$\mathfrak{su}(2) \cong \mathfrak{so}(3)$を示します。
3. [クリフォード代数とSpin(3)](03-spin.md) — 四元数の背後にあるクリフォード代数と$\operatorname{Spin}$群を導入し、$\operatorname{SO}(3)$の2重被覆$\operatorname{Spin}(3)$とスピノルを扱います。パウリ行列が生成するクリフォード代数の偶部分代数として、四元数と$\operatorname{Spin}(3)$を具体的に構成し直します。
4. [Spin(4)とSO(4)](04-spin4.md) — 四元数を4次元空間として左右から独立に挟む作用によって、$\operatorname{SO}(4)$のすべての回転を構成します。二重被覆$\operatorname{Spin}(4) \cong \operatorname{Sp}(1) \times \operatorname{Sp}(1)$が、$C\ell_{3,1}(\mathbb{R}) \cong M_4(\mathbb{R})$と$C\ell_{4,0}(\mathbb{R}) \cong M_2(\mathbb{H})$という2つのクリフォード代数のそれぞれにどう現れるかを見ます。
5. [SO(3)の複素化とSU(3)](05-su3.md) — $\operatorname{SO}(3)$の構造を複素化して$\operatorname{SU}(3)$を構成する、数体系に頼らない初めてのリー群です。ゲルマン行列と$\mathfrak{so}(3) \subset \mathfrak{su}(3)$、構造定数について扱います。
6. [BCH公式と随伴作用](06-bch-adjoint.md) — 指数法則のずれを括弧積の言葉で記述するベイカー・キャンベル・ハウスドルフの公式と、共役作用をリー代数の言葉で捉え直す随伴作用$\operatorname{Ad}/\operatorname{ad}$を扱います。ヤコビ恒等式を導入し、02の同型$\mathfrak{su}(2) \cong \mathfrak{so}(3)$と二重被覆が随伴表現として整理されることを見ます。
7. [八元数とS^7](07-s7.md) — 単位八元数のなす球面$S^7$は、積と逆元を持つ滑らかな球面でありながら、結合法則を欠くためリー群になりません。アルティンの定理により2元までは四元数の世界が生き残ること、結合子として現れるヤコビ恒等式の破れ、ムーファン・ループ、共役作用による$\operatorname{SO}(7)$の生成、ケイリー＝ディクソン構成とフルヴィッツの定理による数体系の系列の終わりを扱います。
8. [八元数の自己同型群G_2](08-g2.md) — 八元数の積の構造を保つ変換の全体として、14次元の例外型リー群$G_2 \subset \operatorname{SO}(7)$を構成します。リー代数が積の微分法則を満たす微分の全体$\operatorname{Der}(\mathbb{O})$であること、結合子による$\operatorname{ad}$の補正、分解$\mathfrak{so}(7) = \mathfrak{g}_2 \oplus \{u \times \cdot\}$、基本三つ組による次元の勘定$14 = 6 + 5 + 3$、虚数単位を固定する部分群としての$\operatorname{SU}(3)$の再登場を扱います。
9. [Spin(7)と三対性](09-spin7-triality.md) — 純虚八元数の左作用が満たすクリフォード関係から$\operatorname{Spin}(7)$を構成し、八元数自身が8次元のスピノルとして半角で回ることを見ます。スピノル$1$の固定部分群としての$G_2$（$S^7 = \operatorname{Spin}(7)/G_2$）、体積要素$L_7$を複素構造とする$\operatorname{Spin}(6) \cong \operatorname{SU}(4)$と$\operatorname{SU}(3)$、倍加構成による$\operatorname{Spin}(8)$、3つの8次元表現を八元数の積が結ぶ三対性と、その対称性の固定部分群としての$G_2$、固定部分群の鎖$28 \to 21 \to 14 \to 8 \to 3 \to 0$を扱います。
10. [ルート系とディンキン図形](10-roots-dynkin.md) — 個々の群の構成から視点を切り替え、コンパクト単純リー代数の分類を俯瞰します。極大トーラスから見た回転数としてのウェイトとその量子化（スピン）、随伴表現のウェイトとしてのルート、整数条件による角度・長さの制限、単純ルートとディンキン図形を整備し、4つの無限系列$A_n, B_n, C_n, D_n$と5つの例外型に至ります。偶然の同型が低ランクの図形の一致であること、三対性が$D_4$の三つ又の対称性であること、$G_2$のルート系が六芒星をなすことを確認し、これまでの群たちを分類表に位置づけ直します。
11. [例外型ジョルダン代数と魔方陣](11-exceptional.md) — 八元数を成分とする$3 \times 3$エルミート行列と対称化した積のなす例外型ジョルダン代数（アルバート代数）を導入し、残る例外型リー群$F_4, E_6, E_7, E_8$を扱います。ジョルダン恒等式が$3 \times 3$まで生き残り$4 \times 4$で破れること、自己同型群$F_4 = \operatorname{Aut}(h_3(\mathbb{O}))$と八元数射影平面$\mathbb{O}P^2 = F_4/\operatorname{Spin}(9)$、三対性の内部化$52 = 28 + 8 + 8 + 8$、複素化の再演による$\mathfrak{e}_6 = \mathfrak{f}_4 \oplus iL$、ティッツの構成とフロイデンタールの魔方陣、$E_7$・$E_8$の次元の勘定を経て、シリーズを総括します。

## 今後の予定

全体構想（[PLAN.md](../PLAN.md)）の「単純リー群の分類と例外型リー群」までを本シリーズの範囲とし、分類（10）と例外型リー群（11）をもって当初の範囲は完結しました。ケイリー＝ディクソン系列との合流点である十六元数と零因子（モレノの定理）は、数体系側の記事[十六元数と零因子](../oct/04-sedenion.md)で扱いました。

### 扱わないこと

本シリーズは入門の入門であり、定理を網羅することは意図していません。多様体としての一般論（ハール測度など）や表現論の一般理論には立ち入らず、分類定理も証明はせずに「分類表が読める」ことを目標とします。また対象はコンパクト群に限り、非コンパクト群（ローレンツ群と$\operatorname{SL}(2,\mathbb C)$など）は時空代数の記事（[PLAN.md](../PLAN.md)の「時空代数とディラック方程式」）に委ねます。

## 記事構成の意図

本シリーズは、単にリー群を羅列するのではなく、以下のような全体を貫く構造的なストーリーを持っています。

- **数体系によるパラレルの終着**: 01と02では、「複素数 $\to \operatorname{SO}(2)$」と「四元数 $\to \operatorname{SO}(3)$」という美しい並行構造を描きます。しかし、この「数体系から直接回転群を作る」という単純なパターンは八元数で破綻します（結合則の喪失によりリー群にならないため）。
- **一般化へのバトンタッチ**: そこで03において、「クリフォード代数」と「$\operatorname{Spin}$群」を導入します。これが高次元の回転群を記述する枠組みへのバトンタッチとなります。続く04では、四元数の「左右の独立した作用」による$\operatorname{SO}(4)$の構成を扱いますが、これはのちの八元数における非結合性への最大の伏線です。
- **八元数編への布石（本文では予告しない）**: 04の「左作用・右作用の行列表現」「$L_uR_v$の数え上げによる$M_4(\mathbb{R}) \cong C\ell_{3,1}$」「射影$P$によるスピノルの行列形式と復元式$L_x = \sum_u R_u(L_xP)R_u^{-1}$」は、八元数編の中核（左作用が生成する$C\ell_{0,6} \cong M_8(\mathbb{R})$、体積要素としての$L_7 = L_1 \cdots L_6$、射影$P$によるスピノルの切り出し、右作用による復元と積の左作用化）にそのまま対応する布石です。04の本文ではあえて予告せず、八元数編の側から「四元数で見た構図の再現」として回収します。
- **拡張の軸の転換**: 数体系の道が途絶えるため、05の $\operatorname{SU}(3)$ では「行列のサイズを上げる」という別の軸へ乗り換えて群を拡張します。
- **壮大な伏線回収**: 後半の07以降では八元数の対称性を扱います。結合律の崩れによって左右の作用が複雑に絡み合い、そこにクリフォード代数が生成する $\operatorname{Spin}(6)$ や $\operatorname{Spin}(7)$ といった群が立ち現れます。そこで $\operatorname{Spin}$ 群の作用を受けるのは八元数自身がなす8次元のスピノルであり、その部分群として前半で構成した $\operatorname{SU}(3)$ や $G_2$ が再登場し、すべての物語が統合されます。
- **分類による俯瞰**: 最後に、個別の群を1つずつ構成してきた視点を切り替え、ルート系とディンキン図形による単純リー群の分類で全体を俯瞰します。ここまでに登場した群が4つの無限系列に収まる一方、分類表には5つの例外型が現れます。$G_2$以外の例外型$F_4, E_6, E_7, E_8$も八元数と深く結びついており、数体系の道が途絶えたはずの場所に、八元数が例外型という形で戻ってくることを見ます。
- **ケイリー＝ディクソン系列との合流**: $G_2$は十六元数の零因子とも接続します（積が$0$になる単位ノルムのペアのなす空間が$G_2$と同相になるというモレノの定理）。本シリーズの例外型リー群と数体系側の[十六元数と零因子](../oct/04-sedenion.md)は、$G_2$を合流点として互いに他方の到達点を照らします。

以上は現時点での構想であり、実際の記事構成は変更される可能性があります。

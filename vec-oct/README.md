# ベクトルから八元数まで

2. [回転：ベクトルから八元数まで](02-rotation.md) - 鏡映の合成として回転を捉え、各代数系での回転の表現を見ていきます

## その他の記事

- [ベクトルの幾何積と指数関数](geometric-product-exp.md) — クリフォード代数における2次元・3次元ベクトルの幾何積を内積と外積の和として導出し、回転を用いた幾何学的解釈を経て、幾何積が指数関数$e^{\theta p}$で表現できることを示します。複素数・四元数との対応関係も確認します。

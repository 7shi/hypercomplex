# その他

四元数・八元数などの主要テーマに含まれない単発記事です。

## 記事

- [指数関数を定積分でマクローリン展開](exp-maclaurin-integral.md) — 指数関数の積分方程式を再帰的に代入し繰り返すことで、マクローリン展開と同じ級数が得られることを示します。
- [エネルギーの量子化とゼータ関数](energy-quantize-zeta.md) — レイリー・ジーンズ、ウィーン、プランクの3つの黒体放射公式が共通の構造を持つことを示し、エネルギーの量子化によって積分が総和に変わる過程と、プランクの公式を全周波数で積分するとゼータ関数$\zeta(4)$が現れる過程を解説します。
- [変数の依存関係と微分](variable-dependence.md) — 従属関係を持つ関数の微分について、先に依存関係を代入してから微分する方法と、全微分を計算してから依存関係を代入する方法が同じ結果になることを確認し、オイラー＝ラグランジュ方程式での偏微分の扱いに応用します。
- [εパラメーターを用いたオイラー＝ラグランジュ方程式](epsilon-euler-lagrange.md) — 経路を$q(t)+\varepsilon\eta(t)$と表して作用積分を$\varepsilon$で微分する方法で、オイラー＝ラグランジュ方程式を導出します。
- [部分積分における関数の微分の利用](integration-by-parts.md) — ライプニッツ則$d(fg)=(df)g+f(dg)$から出発し、関数の微分を用いた表記で部分積分を行うことで、変形の見通しを良くする方法を示します。
- [一般次元における球座標系のバリエーション](spherical-coords.md) — 単位ベクトルの成分を三角関数でどうパターン分けするかによって、一般次元の球座標系に複数のバリエーションが生じることを示し、4次元・6次元の例として量子情報における状態ベクトルを紹介します。
